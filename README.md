# Password-strength-checker

A comprehensive cybersecurity tool that evaluates password security using entropy analysis, pattern detection, crack-time estimation, and real-time feedback.

![Image Alt[](](https://github.com/ahsan-lgtm/password-strength-checker/blob/a8d86291104197317d60f27ed8727570145a0259/PassChecker.png)

- **Password Generator** — Cryptographically secure random passwords with configurable character sets and length (8–128)
- **Check History** — Last 20 analyses saved locally; click any entry to re-analyze
- **Interactive Demo** — Auto-types example passwords on first focus to showcase the tool

## 📖 Usage

### Analyzing a Password
1. Open `index.html` in any browser
2. Click the input field to trigger the interactive demo, or type a password directly
3. The strength meter, score, and all metrics update in real time
4. Review the **Security Criteria** checklist to see which requirements are met
5. Read the **Feedback & Suggestions** panel for specific improvement tips
6. Check **Character Distribution** bars to visualize the composition

### Generating a Password
1. In the **Password Generator** panel, select character sets (A-Z, a-z, 0-9, special)
2. Set the desired length (8–128 characters)
3. Click **Generate** — a cryptographically secure password appears
4. Click **Copy** to copy it to your clipboard

### Using History
- Previously analyzed passwords appear in the **Check History** panel
- Click any history entry to reload it for re-analysis
- Use **Clear History** to remove all saved entries
